Module WeldJoint

A weld joint essentially glues two bodies together.

A weld joint ma y distort somewhat because the island constraint solver is approximate.

Type WeldJoint

WeldJoint:getAnchorA()

Returns the anchor point on bodyA in world coordinates.
WeldJoint:getAnchorB()

Returns the anchor point on bodyB in world coordinates.
WeldJoint:getBodyA()

Returns the first body attached to this joint.
WeldJoint:getBodyB()

Returns the second body attached to this joint.
WeldJoint:getType()

Returns the type of the concrete joint.
WeldJoint:isActive()

Short-cut function to determine if either body is inactive.

Type WeldJoint

Field(s)

WeldJoint:getAnchorA()

Returns the anchor point on bodyA in world coordinates.

Return value

The x and y coordinates of the anchor point

WeldJoint:getAnchorB()

Returns the anchor point on bodyB in world coordinates.

Return value

The x and y coordinates of the anchor point

WeldJoint:getBodyA()

Returns the first body attached to this joint.

Return value

The first body attached to this joint

WeldJoint:getBodyB()

Returns the second body attached to this joint.

Return value

The second body attached to this joint

WeldJoint:getType()

Returns the type of the concrete joint.

The return value can be one of the b2.REVOLUTE_JOINT, b2.PRISMATIC_JOINT, b2.DISTANCE_JOINT, b2.PULLEY_JOINT, b2.MOUSE_JOINT, b2.GEAR_JOINT, b2.LINE_JOINT, b2.WELD_JOINT, b2.FRICTION_JOINT.

Return value

The type of the concrete joint

WeldJoint:isActive()

Short-cut function to determine if either body is inactive.

Return value

true if both bodyA and bodyB is active, false otherwise
