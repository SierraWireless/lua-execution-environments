Body A rigid body.

EventDispatcher All classes that dispatch events inherit from EventDispatcher.

FontBase FontBase is the base class for Font and TTFont classes.

Geolocation The Geolocation class inherits from the following class: EventDispatcher

The Geolocation class is used to configure the parameters and dispatching of location and heading related events.

Joint The b2.Joint class is the base joint class.
Matrix The Matrix class specifies 2D transformation from one coordinate space to another.
MouseJoint A mouse joint is used to make a point on a body track a specified world point.
MovieClip The MovieClip class inherits from the following classes: Sprite > EventDispatcher.
PolygonShape A convex polygon.
PrismaticJoint A prismatic joint.
PulleyJoint The pulley joint is connected to two bodies and two fixed ground points.
RevoluteJoint A revolute joint constrains two bodies to share a common point while they are free to rotate about the point.
Shape The Shape class is used create and display vector graphics.
Sound The Sound class lets you load and play WAV or MP3 sound files.
SoundChannel The SoundChannel class is used to control and monitor a playing sound.
Sprite The Sprite class is the base class for all objects that can be placed on the scene tree.
Stage The Stage class represents the top of the scene tree hierarchy.
TTFont The TTFont class is used to load true type fonts.
TextField The TextField class is used to create display objects for text display.
Texture The Texture class lets you work with textures in an application.
TextureBase TextureBase is the base class for Texture and TexturePack classes.
TexturePack The TexturePack class specifies a texture pack (or texture atlas).
TextureRegion The TextureRegion class specifies a texture and a rectangular region in it.
TileMap The TileMap class is used to work with tile maps easily and efficiently.
Timer The Timer class is used to execute a code at specified intervals.
WeldJoint A weld joint essentially glues two bodies together.
WheelJoint A wheel joint provides two degrees of freedom: translation along an axis fixed in bodyA and rotation in the plane.
World The b2.World class inherits from the following class: EventDispatcher.
b2 To load the Box2D library, call require "box2d".
b2.DebugDraw The b2.DebugDraw class is used to provide debug drawing of physical entities in your application.
b2.GearJoint A gear joint is used to connect two joints together.
