Module Timer

The Timer class is used to execute a code at specified intervals.
The listener functions are registered through Event.TIMER and Event.TIMER_COMPLETE events.

### Timer Events * Event.TIMER = "timer" * Event.TIMER_COMPLETE = "timerComplete"

Type Timer

Timer.delayedCall(delay, func, data) Provides a simple way to call a function after a set amount of time.
Timer:getCurrentCount() Returns the current trigger count of the timer.
Timer:getDelay() Returns the time interval between timer events in milliseconds.
Timer:getRepeatCount() Returns the number of repetitions the timer will make.
Timer:isRunning() Returns the current running status of timer.
Timer.new(delay, repeatCount) Creates a new Timer object with the specified delay and repeatCount states.
Timer.pauseAll() Pause all timers.
Timer.pauseAllTimers() **This function is obsolete.
Timer:reset() Stops the timer and sets the currentCount property to 0.
Timer.resumeAll() Resume all timers.
Timer.resumeAllTimers() **This function is obsolete.
Timer:setDelay(delay) Sets the time interval between timer events in milliseconds.
Timer:setRepeatCount(repeatCount) Sets the number of repetitions the timer will make.
Timer:start() Starts the timer.
Timer:stop() Stops the timer.
Timer.stopAll() Stop all timers.

Type Timer

Field(s)

Timer.delayedCall(delay, func, data)
Provides a simple way to call a function after a set amount of time.

Parameters

  • delay : (number) Delay in miliseconds before the function is called
  • func : (function) Function to call
  • data : (optional) An optional data parameter that is passed as a first argument to the function
Timer:getCurrentCount()
Returns the current trigger count of the timer.
It starts with 0 and if it reaches repeatCount value, timer stops.

Return value

The current trigger count.

Timer:getDelay()
Returns the time interval between timer events in milliseconds.

Return value

The time interval between timer events in milliseconds.

Timer:getRepeatCount()
Returns the number of repetitions the timer will make.
A value of 0 means the timer runs infinitely. If nonzero, the timer runs the specified number of times and then stops.

Return value

The number of repetitions.

Timer:isRunning()
Returns the current running status of timer.

Return value

true if the timer is running, false otherwise.

Timer.new(delay, repeatCount)
Creates a new Timer object with the specified delay and repeatCount states.

Parameters

  • delay : The time interval between timer events in milliseconds.
  • repeatCount : (default = 0) The number of repetitions. A value of 0 runs the timer infinitely. If nonzero, the timer runs the specified number of times and then stops.

Return value
