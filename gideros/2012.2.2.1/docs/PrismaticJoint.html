Module PrismaticJoint

A prismatic joint.
This joint provides one degree of freedom: translation along an axis fixed in body1. Relative rotation is prevented. You can use a joint limit to restrict the range of motion and a joint motor to drive the motion or to model joint friction.

Type PrismaticJoint

PrismaticJoint:enableLimit(flag) Enables or disables the joint limit.
PrismaticJoint:enableMotor(flag) Enables or disables the joint motor.
PrismaticJoint:getAnchorA() Returns the anchor point on bodyA in world coordinates.
PrismaticJoint:getAnchorB() Returns the anchor point on bodyB in world coordinates.
PrismaticJoint:getBodyA() Returns the first body attached to this joint.
PrismaticJoint:getBodyB() Returns the second body attached to this joint.
PrismaticJoint:getJointSpeed() Returns the current joint translation speed, usually in meters per second.
PrismaticJoint:getJointTranslation() Returns the current joint translation, usually in meters.
PrismaticJoint:getLimits() Returns the lower and upper joint limit, usually in meters.
PrismaticJoint:getMotorSpeed() Returns the motor speed in meters per second.
PrismaticJoint:getType() Returns the type of the concrete joint.
PrismaticJoint:isActive() Short-cut function to determine if either body is inactive.
PrismaticJoint:isLimitEnabled() Is the joint limit enabled?
PrismaticJoint:isMotorEnabled() Is the joint motor enabled?
PrismaticJoint:setLimits(lower, upper) Sets the joint limits, usually in meters.
PrismaticJoint:setMaxMotorForce(force) Sets the maximum motor force, usually in N.
PrismaticJoint:setMotorSpeed(speed) Sets the motor speed in meters per second.
