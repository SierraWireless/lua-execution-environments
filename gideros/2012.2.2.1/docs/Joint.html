Module Joint

The b2.Joint class is the base joint class.
Joints are used to constraint two bodies together in various fashions. Some joints also feature limits and motors.

Type Joint

Joint:getAnchorA() Returns the anchor point on bodyA in world coordinates.
Joint:getAnchorB() Returns the anchor point on bodyB in world coordinates.
Joint:getBodyA() Returns the first body attached to this joint.
Joint:getBodyB() Returns the second body attached to this joint.
Joint:getType() Returns the type of the concrete joint.
Joint:isActive() Short-cut function to determine if either body is inactive.

Type Joint

Field(s)

Joint:getAnchorA()
Returns the anchor point on bodyA in world coordinates.

Return value

The x and y coordinates of the anchor point

Joint:getAnchorB()
Returns the anchor point on bodyB in world coordinates.

Return value

The x and y coordinates of the anchor point

Joint:getBodyA()
Returns the first body attached to this joint.

Return value

The first body attached to this joint

Joint:getBodyB()
Returns the second body attached to this joint.

Return value

The second body attached to this joint

Joint:getType()
Returns the type of the concrete joint.
The return value can be one of the b2.REVOLUTE_JOINT, b2.PRISMATIC_JOINT, b2.DISTANCE_JOINT, b2.PULLEY_JOINT, b2,MOUSE_JOINT, b2.GEAR_JOINT, b2.LINE_JOINT, b2.WELD_JOINT, b2.FRICTION_JOINT.

Return value

The type of the concrete joint

Joint:isActive()
Short-cut function to determine if either body is inactive.

Return value

true if both bodyA and bodyB is active, false otherwise
