Module RevoluteJoint

A revolute joint constrains two bodies to share a common point while they are free to rotate about the point.
The relative rotation about the shared point is the joint angle. You can limit the relative rotation with a joint limit that specifies a lower and upper angle. You can use a motor to drive the relative rotation about the shared point. A maximum motor torque is provided so that infinite forces are not generated.

Type RevoluteJoint

RevoluteJoint:enableLimit(flag) Enables or disables the joint limit.
RevoluteJoint:enableMotor(flag) Enables or disables the joint motor.
RevoluteJoint:getAnchorA() Returns the anchor point on bodyA in world coordinates.
RevoluteJoint:getAnchorB() Returns the anchor point on bodyB in world coordinates.
RevoluteJoint:getBodyA() Returns the first body attached to this joint.
RevoluteJoint:getBodyB() Returns the second body attached to this joint.
RevoluteJoint:getJointAngle() Returns the current joint angle in radians.
RevoluteJoint:getJointSpeed() Returns the current joint angle speed in radians per second.
RevoluteJoint:getLimits() Returns the lower and upper joint limit in radians.
RevoluteJoint:getMotorSpeed() Returns the motor speed in radians per second.
RevoluteJoint:getType() Returns the type of the concrete joint.
RevoluteJoint:isActive() Short-cut function to determine if either body is inactive.
RevoluteJoint:isLimitEnabled() Is the joint limit enabled?
RevoluteJoint:isMotorEnabled() Is the joint motor enabled?
RevoluteJoint:setLimits(lower, upper) Sets the joint limits in radians.
RevoluteJoint:setMaxMotorTorque(torque) Sets the maximum motor torque, usually in N-m.
RevoluteJoint:setMotorSpeed(speed) Sets the motor speed in radians per second.
