Module DistanceJoint

A distance joint constrains two points on two bodies to remain at a fixed distance from each other.
You can view this as a massless, rigid rod.

Type DistanceJoint

DistanceJoint:getAnchorA() Returns the anchor point on bodyA in world coordinates.
DistanceJoint:getAnchorB() Returns the anchor point on bodyB in world coordinates.
DistanceJoint:getBodyA() Returns the first body attached to this joint.
DistanceJoint:getBodyB() Returns the second body attached to this joint.
DistanceJoint:getDampingRatio() Returns the damping ratio of this distance joint.
DistanceJoint:getFrequency() Returns the mass-spring-damper frequency of this distance joint in Hertz.
DistanceJoint:getLength() Returns the length of this distance joint, usually in meters.
DistanceJoint:getType() Returns the type of the concrete joint.
DistanceJoint:isActive() Short-cut function to determine if either body is inactive.
DistanceJoint:setDampingRatio(ratio) Sets the damping ratio of this distance joint.
DistanceJoint:setFrequency(frequency) Sets the mass-spring-damper frequency in Hertz.
DistanceJoint:setLength(length) Sets the natural length.

DistanceJoint:getType()
Returns the type of the concrete joint.
The return value can be one of the b2.REVOLUTE_JOINT, b2.PRISMATIC_JOINT, b2.DISTANCE_JOINT, b2.PULLEY_JOINT, b2,MOUSE_JOINT, b2.GEAR_JOINT, b2.LINE_JOINT, b2.WELD_JOINT, b2.FRICTION_JOINT.

Return value

The type of the concrete joint

DistanceJoint:isActive()
Short-cut function to determine if either body is inactive.

Return value

true if both bodyA and bodyB is active, false otherwise

DistanceJoint:setDampingRatio(ratio)
Sets the damping ratio of this distance joint.
0 = no damping, 1 = critical damping.

Parameter

  • ratio : (number) the damping ratio
DistanceJoint:setFrequency(frequency)
Sets the mass-spring-damper frequency in Hertz.

Parameter

  • frequency : (number) the mass-spring-damper frequency in Hertz
DistanceJoint:setLength(length)
Sets the natural length.
Manipulating the length can lead to non-physical behavior when the frequency is zero.

Parameter

  • length : (number) the length of this distance joint, usually in meters.
