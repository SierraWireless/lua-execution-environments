Module Font

The 'Font' class is used to load fonts created by "Gideros Font Creator".

Gideros Font Creator exports two files: A .txt file that specifes the positions of character glyph a .png file of font. Use these two files to create Font object as:

local font = Font.new("font.txt", "font.png")

Font.new(txtfile, imagefile, filtering) Creates a new Font object.
