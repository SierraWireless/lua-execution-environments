Module FrictionJoint

Friction joint.
This is used for top-down friction. It provides 2D translational friction and angular friction.

Type FrictionJoint

FrictionJoint:getAnchorA() Returns the anchor point on bodyA in world coordinates.
FrictionJoint:getAnchorB() Returns the anchor point on bodyB in world coordinates.
FrictionJoint:getBodyA() Returns the first body attached to this joint.
FrictionJoint:getBodyB() Returns the second body attached to this joint.
FrictionJoint:getMaxForce() Returns the maximum friction force in N.
FrictionJoint:getMaxTorque() Returns the maximum friction torque in N*m.
FrictionJoint:getType() Returns the type of the concrete joint.
FrictionJoint:isActive() Short-cut function to determine if either body is inactive.
FrictionJoint:setMaxForce(force) Sets the maximum friction force in N.
FrictionJoint:setMaxTorque(torque) Sets the maximum friction torque in N*m.
