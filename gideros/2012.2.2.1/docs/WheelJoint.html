Module WheelJoint

A wheel joint provides two degrees of freedom: translation along an axis fixed in bodyA and rotation in the plane.
You can use a joint limit to restrict the range of motion and a joint motor to drive the rotation or to model rotational friction. This joint is designed for vehicle suspensions.

Type WheelJoint

WheelJoint:Joint() Returns the anchor point on bodyB in world coordinates.
WheelJoint:enableMotor(flag) Enables or disables the joint motor.
WheelJoint:getJointSpeed() Returns the current joint translation speed in meters per second.
WheelJoint:getJointTranslation() Returns the current joint translation in meters.
WheelJoint:getMaxMotorTorque() Returns the maximum motor torque in N-m.
WheelJoint:getMotorSpeed() Returns the motor speed in radians per second.
WheelJoint:getSpringDampingRatio() Returns the spring damping ratio.
WheelJoint:getSpringFrequencyHz() Returns the spring frequency in hertz.
WheelJoint:isActive() Short-cut function to determine if either body is inactive.
WheelJoint:isMotorEnabled() Is the joint motor enabled?
WheelJoint:setMaxMotorTorque(torque) Sets the maximum motor torque in N-m.
WheelJoint:setMotorSpeed(speed) Sets the motor speed in radians per second.
WheelJoint:setSpringDampingRatio(damping) Sets the spring damping ratio.
WheelJoint:setSpringFrequencyHz(frequency) Sets the spring frequency in hertz.
