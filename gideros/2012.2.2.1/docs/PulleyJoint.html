Module PulleyJoint

The pulley joint is connected to two bodies and two fixed ground points.

The pulley supports a ratio such that: length1 + ratio * length2 <= constant Yes, the force transmitted is scaled by the ratio. The pulley also enforces a maximum length limit on both sides. This is useful to prevent one side of the pulley hitting the top.

Type PulleyJoint

PulleyJoint:getAnchorA()

Returns the anchor point on bodyA in world coordinates.
PulleyJoint:getAnchorB()

Returns the anchor point on bodyB in world coordinates.
PulleyJoint:getBodyA()

Returns the first body attached to this joint.
PulleyJoint:getBodyB()

Returns the second body attached to this joint.
PulleyJoint:getGroundAnchorA()

Returns the x and y coordinates of the first ground anchor.
PulleyJoint:getGroundAnchorB()

Returns the x and y coordinates of the second ground anchor.
PulleyJoint:getLengthA()

Returns the current length of the segment attached to bodyA.
PulleyJoint:getLengthB()

Returns the current length of the segment attached to bodyB.
PulleyJoint:getRatio()

Returns the pulley ratio.
PulleyJoint:getType()

Returns the type of the concrete joint.
PulleyJoint:isActive()

Short-cut function to determine if either body is inactive.
