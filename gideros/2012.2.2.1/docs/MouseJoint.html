Module MouseJoint

A mouse joint is used to make a point on a body track a specified world point.

This a soft constraint with a maximum force. This allows the constraint to stretch and without applying huge forces.

Type MouseJoint

MouseJoint:getAnchorA()

Returns the anchor point on bodyA in world coordinates.
MouseJoint:getAnchorB()

Returns the anchor point on bodyB in world coordinates.
MouseJoint:getBodyA()

Returns the first body attached to this joint.
MouseJoint:getBodyB()

Returns the second body attached to this joint.
MouseJoint:getDampingRatio()

Returns the damping ratio.
MouseJoint:getFrequency()

Returns the response speed in Hertz.
MouseJoint:getMaxForce()

Returns the maximum force in Newtons.
MouseJoint:getTarget()

Returns the x and y coordinates of the target point.
MouseJoint:getType()

Returns the type of the concrete joint.
MouseJoint:isActive()

Short-cut function to determine if either body is inactive.
MouseJoint:setDampingRatio(ratio)

Sets the damping ratio.
MouseJoint:setFrequency(frequency)

Sets the response speed in Hertz.
MouseJoint:setMaxForce(force)

Sets the maximum force in Newtons.
MouseJoint:setTarget(x, y)

Updates the target point.

MouseJoint:getType()

Returns the type of the concrete joint.

The return value can be one of the b2.REVOLUTE_JOINT, b2.PRISMATIC_JOINT, b2.DISTANCE_JOINT, b2.PULLEY_JOINT, b2,MOUSE_JOINT, b2.GEAR_JOINT, b2.LINE_JOINT, b2.WELD_JOINT, b2.FRICTION_JOINT.

Return value

The type of the concrete joint
